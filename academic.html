PROJECTS
Library Management
It is a project developed in C++ Programming Language. It displays a menu for user to choose between issue a book , submit a book , view available books and view details of issued book. It update the available books ﬁle as soon as any issue or submition of book occurs. It extract data from ﬁle and store the updated data in ﬁle again.
Contact Management
It is a project developed in C++ Programming Language. It provides a user menu to choose between view contacts , add contacts , delete contacts and update contacts . It provides alphabetical viewing of all contacts. It extract data from ﬁle as per user choose and store the same in ﬁle after updation.
Forget Password
It is a login page created using PHP , HTML , MySQL and CSS . It stores the sign up details in database . While login it match the details with the associated database and based on it's correction it move to next page . In case of forget password it sends your password to you through email.
Hotel Booking
It is a project created using Java and MySQL. It shows the availability of hotel rooms in a hotel . We can book room , cancel room and order food . It stores all the information in database and retrieve the same on request. At the time of check out it displays the overall bill of our stay in hotel.
Online bus booking
It is a website created with PHP , JavaScript, CSS , HTML and MySQL. The ask the user to login to book tickets. User can choose the departure and destination place from the drop down list along with date and number of seats required . It then show the availability of buses along with seat left and price after retrieving from database . Then user can book seat and proceed towards payment. Different payment methods are available for payment. User can also see his past trips after login.
